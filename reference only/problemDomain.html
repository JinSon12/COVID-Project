Overview of the Topic

Why
COVID?

COVID outbreak is a pandemic, which is causing economical, personal issues around the world.
Currently, there are more than a million of global population diagnosed with the virus.
There are a lot of web pages providing information regarding COVID.
Most of them are in website form, giving users a general overview of the COVID statistics around the world.

I wanted to use the same technology to provide state-specific information, so that people in each state can view more specific or detailed statistics regarding their own state. I found that most of the web pages provide more global information; information more specific to countries.

Couple of them would include :

CoronaVirus COVID-19 Global Cases ARCGIS  
CDC COVID  

However, I wanted the users to give freedom of accesing more state-specific information

          So this app has following functions:

          1) A table containing state-wide information for the date of the access
          2) Click on each rows to get more information about a specific state
          3) Detailed information would give the users

  • 10 day overview of the data of the state in the form of graph
  • Some useful local news feed
  • Guidelines for safe practice

User Case

Dan has a family in Washington State, and he wants to know more about the current situation of COVID outbreak in his State, with the daily changes in the number of patients, deaths, recovered and so on. He also wants to view the trends of these number for a ten day span presented in a visualization. This way, he can see the trends in the infection status of his state. He wants to also know the local news feed as well. He can use the functions of the web app to retrieve the information he wants. Using these functions, he would be able to guage on the current situation of the COVID in his state more efficiently, and can develop plans accordingly.
