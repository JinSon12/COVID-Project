Feature 1: Country-Wide Statistics

This is going to be the landing of the web page.
Main feature of this page is a table that displays the country wide information of COVID in United States for the date and time of access. This will inlcude the data from all 50 states in US. The data will include the number of diagnosed, death, recovered in each state. Users can click on to refresh the statistics. Users can also click on the row of the state they are interested in and then go view a more detailed statistics for the state of their interest.
The data will be fetched from The COVID Tracking Project API.

Feature 2: State-Wide Statistics

The users can scroll down and see the state-wide statistics.

Initially, this will be showing WA. Again, the user would be able to view the information of the state they are interested in by clicking on a row (state) of the table in the landing section. This is going to be the a section with a more detailed statistics of COVID in the state of user's choice.

Main feature of this page is a line graph that displays the trend. The users would be able to select which data to show by using the checkbox below the chart.
Users can click on checkboxese to include the data of their choice.

Feature 3: News Feed

This is going to be the a section for displaying relavant news about COVID.
These will include sources from YouTube and google Search. The users would be able to click on each article, and they will be directed to the webpage where they can read the news in detail.
